Heap Sort Algorithm

Heap Sort is a comparison-based sorting algorithm that uses a binary heap data structure to build a heap and efficiently find the maximum (or minimum) element and repeatedly extract it from the heap. It has a time complexity of O(n log n) for the worst, average, and best cases.

Heap sort is an efficient and popular sorting algorithm. The heap sort concept is to "eliminate" the element from the heap part of the list one-by-one and insert them to the sorted part of the list. Before learning more about the heap sorting algorithm, let's discuss the heap data structure.

It is an in-place algorithm, which means a fixed amount of memory is used to store the sorted list, or the memory size doesn't rely on the size of the preliminary list

Working

Build Heap:

  • Build a binary heap from the input data.
  • The heap is a complete binary tree where the value of each node is greater than or equal to the values of its children.

Heapify:

  • Repeatedly remove the largest (for max heap) element from the heap.
  • Swap it with the last element in the heap (putting it in its final position).

Heap Size Reduction:

  • Reduce the size of the heap by one.
  • Repeat steps 2 and 3 until the heap is empty.
